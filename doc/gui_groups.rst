Manage groups on the model
==========================

.. _guiaddgroup:

Add group
=========

A group is characterized by:

- its name,
- its type:
      - HexaCell (default)
      - EdgeCell
      - QuadCell
      - HexaNode
      - QuadNode
      - EdgeNode
      - VertexNode
- its elements.

To define groups on the model in the **Main Menu** select **Groups -> Add Group**.

   :align: center

Remove group
============

To remove a group from the model in the **Main Menu** select **Groups -> Remove Group**.
