.. _guidisconnectelements:

===================
Disconnect elements
===================

To disconnect elements in the **Main Menu** select **Model -> Disconnect**.

.. _guidiscoquad:

Disconnect a quadrangle
=======================

**Arguments:**

- q: the quadrangle to disconnect.
- from hexa: the hexahedron to disconnect the quadrangle from.
